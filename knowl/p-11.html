  • Added terms are simplified using coefficients. For example, \(x + x + x\) can be simplified as \(3\cdot x\) (where 3 is the coefficient).
  • Multiplied terms are simplified using exponents. For example, \(x \cdot x \cdot x\) is represented as \(x^3\)
  • Like terms are added together, for example, \(2\cdot x^2 + 3\cdot a\cdot b - x^2 + a\cdot b\) is written as \(x^2 + 4\cdot a\cdot b\text{,}\) because the terms containing \(x^2\) are added together, and, the terms containing \(a\cdot b\) are added together.
  • Brackets can be "multiplied out", using distributivity. For example, \(x\cdot (2x + 3)\) can be written as \((x \cdot 2x) + (x \cdot 3)\) which can be written as \(2x^2 + 3x\text{.}\)
  • Expressions can be factored. For example, \(6x^5 + 3x^2\text{,}\) by dividing both terms by \(3x^2\) can be written as \(3x^2\cdot (2x^3 + 1)\)
